Lecture 4: Lists, Links, and Images

Ordered Lists

Ordered lists display items in a specific numbered sequence. These are typically used when the order matters, such as step-by-step instructions or ranking items.

Example:

  1. Download the installer
  2. Open the setup file
  3. Follow the installation wizard
  4. Finish and launch the application

You can also change how the numbering appears using CSS.

Styled Example:

  1. Prepare materials
  2. Assemble components
  3. Test functionality
  4. Finalize design
  5. Deploy product

Unordered Lists

Unordered lists show items without any specific sequence. They’re useful when listing features, options, or any information where order doesn’t matter.

Example:

  • Keyboard
  • Mouse
  • Monitor
  • Speaker

Like ordered lists, the bullet styles in unordered lists can be customized using CSS.

Styled Example:

  • Apples
  • Bananas
  • Cherries
  • Dates

Description Lists

Description lists are used to pair terms with their definitions. They use the <dl> element, with <dt> for the term and <dd> for the description.

Example:

Laptop
A portable personal computer designed for mobile use.
SSD
Solid State Drive — a faster alternative to traditional hard drives.

Nested Lists

Nested lists are lists within a list, often used to organize content into subcategories.

Example:

  • Languages
    • Python
    • JavaScript
  • Frameworks
    • React
    • Vue

Horizontal Lists

With CSS, lists can be displayed in a horizontal layout, which is commonly used for navigation menus.

Example:

  • Google
  • Facebook
  • Twitter

Hyperlinks (Links)

Links, also known as hyperlinks, allow users to navigate from one page to another. They are created using the <a> tag and can be either text or images.

Types of Links:

  • Absolute URL: A full URL pointing to an external site (e.g., Visit Example).
  • Relative URL: A local link to another page in the same site (e.g., Lecture 3: Tables).
  • Anchor Link: A link to a specific section on the same page (e.g., Jump to Ordered Lists).

Target Attributes:

  • _blank — Opens the link in a new tab or window.
  • _self — Opens the link in the same tab (default behavior).
  • _parent — Opens the link in the parent frame (if applicable).
  • _top — Opens the link in the full window, ignoring frames.

Example Navigation:

Images as Links

You can also use images as clickable links. This is often used in buttons or image-based menus.

HTML Images

Images are added using the <img> tag. You must specify at least the src (source) and alt (alternative text) attributes.

<img src="image.jpg" alt="Description" width="300" height="200">
